Commenting in Design Review & Generating PDF for review

Last updated on : 23-12-2017

Design Review is the free software by Autodesk. Which, is created to review documents, add comments, and add stamps. Design review takes a lot of document formats. Some of the most used are .dwg , .pdf, .jpg. You can import your AuotCAD drawings, PDF documents, and images in Design Review to comment.

Before we start, Make sure Design Review & A PDF creator is installed on your system. PDF Architect or doPDF are two creators used here at MSA. Check in the start menu if any of them are installed on your system.

How to export from AutoCAD
  • Make sure you have your layout properly set up for plotting. To know more about this, please check this tutorial. If you have more than one sheets in this file and you wish to send all the sheets for review, you are expected to send a combined DWFX file. You can check this tutorial to know how to generate a document with multiple sheets.
  • Enter keyboard shortcut Ctrl+P from Layout tab in AutoCAD.
  • Change the Plotter name to DWFx as shown in the image below
  • Choose location where you want to save the file
  • Double click on that file. It'll open in Design Review
